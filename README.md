# ATL
ATL is a collection of links to use for any browser where you can access GitHub, but not a search engine easily. it is designed to make it more convenient to access things you'd like to without a search engine.
# What does ATL mean?
ATL is an acronym with two meanings. Auth To Link and At The Library. The second meaning comes from an image.
